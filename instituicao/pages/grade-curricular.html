Desenvolvimento Web Full Stack

6 meses · 360h
📋 Grade Curricular — Desenvolvimento Web Full Stack
Módulo Disciplina Nível C/H (h) Resumo da Ementa
Módulo 1
Fundamentos		 HTML5 Semântico e Acessibilidade Básico 20h Estrutura semântica, tags HTML5, ARIA, boas práticas de acessibilidade.
CSS3, Flexbox e Grid Layout Básico 24h Box model, layouts responsivos, animações e transições CSS.
JavaScript ES6+ e TypeScript Médio 40h Fundamentos JS, ES6+, tipagem estática com TypeScript, DOM manipulation.
Módulo 2
Front-End		 React.js com Hooks e Context API Médio 48h Componentes, estado, ciclo de vida, hooks customizados, gerenciamento de estado.
Estilização Avançada (Styled Components, Tailwind) Médio 20h CSS-in-JS, utility-first CSS, design systems e temas.
Módulo 3
Back-End		 Node.js, Express e REST APIs Médio 48h Servidor HTTP, rotas, middlewares, autenticação JWT, documentação Swagger.
Banco de Dados SQL e NoSQL Médio 32h PostgreSQL, MongoDB, ORM com Prisma, modelagem e queries avançadas.
Git, GitHub e Deploy em Nuvem Avançado 24h Versionamento, pull requests, CI/CD básico, deploy na Vercel e Railway.
🏆 Projeto Final — Portfólio Full Stack Avançado 104h Desenvolvimento de aplicação completa com front-end, back-end e banco de dados.
CARGA HORÁRIA TOTAL: 360h 8 disciplinas + projeto final
🟩 Básico: sem pré-requisitos 🟨 Médio: requer módulo anterior 🟪 Avançado: requer módulos 1 e 2

Python & Inteligência Artificial

5 meses · 300h
📋 Grade Curricular — Python & Inteligência Artificial
Módulo Disciplina Nível C/H (h) Resumo da Ementa
Módulo 1
Python Base		 Fundamentos de Python Básico 30h Sintaxe, tipos de dados, estruturas de controle, funções e módulos.
Estruturas de Dados e Algoritmos Básico 24h Listas, dicionários, pilhas, filas, complexidade de algoritmos.
Pandas, NumPy e Matplotlib Médio 32h Manipulação de dados tabulares, operações vetoriais e visualização.
Módulo 2
Machine Learning		 Machine Learning com Scikit-learn Médio 40h Regressão, classificação, clustering, validação cruzada e métricas.
Redes Neurais e Deep Learning Avançado 48h TensorFlow, Keras, CNNs, RNNs, transfer learning e fine-tuning.
Módulo 3
IA Aplicada		 Processamento de Linguagem Natural (NLP) Avançado 36h Tokenização, embeddings, transformers, BERT e integração com LLMs.
Projetos práticos com APIs de IA Avançado 40h OpenAI API, LangChain, chatbots, automação inteligente e deploy.
🏆 Projeto Final — Aplicação de IA Avançado 50h Desenvolvimento de solução completa com IA integrada e interface web.
CARGA HORÁRIA TOTAL: 300h 7 disciplinas + projeto final
🟩 Básico: sem pré-requisitos 🟨 Médio: requer módulo anterior 🟪 Avançado: requer módulos 1 e 2
